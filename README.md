# Secure AI Accelerator with Hardware-Assisted Adversarial Attack Detection

## Overview

This project presents a secure AI accelerator implemented using Verilog HDL and verified using Xilinx Vivado.

## Features

- 2×2 Systolic Array
- MAC-based Processing Elements
- Score Selector
- Hardware Attack Detector
- RTL Simulation and Verification

## Tools Used

- Verilog HDL
- Xilinx Vivado 2018.2
